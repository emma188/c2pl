Please use intellij to run the whole project
before running, you need to download sqlite and create 4 database:
data1.db, data2.db, data3.db central.db
There are 2 main function in CentralSite and DataSite.
Using intellij to run main in CentralSite first then run main in DataSite.
There are four test. In the main function of DataSite,
there is a variable named testNum. you can change it from 1 to 4.
please note for every test, you need to run CentralSite and DataSite again.
